Simulación de ARM100 con Mujoco

Belén Bravo, Andrés Geisee, Pablo Mesa
Universidad de los Andes
2025
Code

Abstract

En el proyecto se busca entrenar la simulación de un brazo robótico del tipo ARM100 para que sea capaz de doblar un objeto con material similar a la tela. Para poder cumplir este objetivo, se utilizó el programa Mujoco para poder simular distintas situaciones con el brazo robótico.

Creación del objeto y ARM100 con XML

Para la creación de los objetos con XML, se elige el ancho, altura y longitud del cubo y la densidad de este, y se incorpora a la escena. Para el brazo, se utilizó un archivo del modelo 3D del brazo robótico ARM1OO, el cual se incorporó a un archivo XML, al cual se le agregaron las 6 articulaciones del brazo robótico para poder controlarlo mediante el código.

Movimiento del brazo robótico y manipulación de objetos.

Para el movimiento del brazo robótico, se utilizan seis distintas coordenadas: la rotación de la base, el ángulo de esta, el ángulo del codo, el ángulo de la muñeca, la rotación de la muñeca y la apertura de la mandíbula del brazo robótico. Para poder levantar el cubo, se programan cinco posiciones: sobre el objeto, el brazo robótico cerrando la mandíbula alrededor del objeto, levantado el objeto, transladándolo sobre la caja, y finalmente soltándolo para que entre en la caja.
